Cloudie

Cloudie uses LoRa sensors (Dragino LHT65, Arduino MKR WAN 1310 with MKR ENV Shield) to measure temperature, pressure, humidity, and light. The data is transmitted via LoRaWAN to The Things Network (TTN), processed through an MQTT broker, and securely stored in an SQL database on Microsoft Azure. This system ensures reliable, real-time weather monitoring.

Vault

This Arduino-based vault system features a servo motor lock, a rotary encoder for passcode input, and 7-segment displays for real-time feedback. An OLED screen provides additional status updates. The passcode is stored in Arduino’s memory, ensuring persistence after power loss. After three failed attempts, the system triggers a timeout with a buzzer and LED alert, enhancing security.
